#FiddleGrabberFusion


## Dependencies
fs
mkdirp
phantomjs



## Usage Guide
### Installing

For using this packge u need to install phantomjs fs and mkdirp as follows

```
 npm install phantomjs
 npm install fs
 npm install mkdirp
```

After installing the package you can use the package as follows
```
 phantomjs fetch.js
```
